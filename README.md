# OpenGL_Model_Loader
OpenGL based software to read .OBJ with custom shader that adds snow to the surface of the model based on the fall direction

-- Features:
- specular lighting calculation.
- normal mapping.
- snow accumulation based on direction.

# Controls:
- WASD move.
- LeftAlt focus.
- P snow.
- R reset.
